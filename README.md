# 2026 Happy Yuan Day - 媛來接力

這是一款由粉絲製作的網頁遊戲，用來慶祝Yuan (媛媛)，來自GENBLUE (幻藍小熊)團體的生日。

這款遊戲是一個有趣的互動體驗，玩家幫助「小媛寶」(小媛寶)接住物品以獲取分數，並回答關於媛的測驗問題。該專案還設有全球排行榜和個人成就獎勵。

## 遊戲玩法

1.  **開始遊戲**：在瀏覽器中打開`index.html`檔案。
2.  **控制小媛寶**：使用左右方向鍵（或行動裝置上的螢幕按鈕）來移動角色。
3.  **接住物品**：不同物品會從螢幕頂部掉落。
    *   **正面物品**（心形、餅乾、吉他）：接住這些來增加分數。
    *   **負面物品**（燒焦餅乾、鬧鐘）：避免接住，因為會扣分。
    *   **特殊物品**（應援棒）：帶來高分。
    *   **問題物品**：接住後會觸發測驗問題。
4.  **回答測驗**：當出現測驗時，選擇正確答案以獲得額外分數，答錯會扣分。
5.  **熱潮時間**：接住正面物品會累積「熱潮」值，滿值時啟動熱潮時間，短時間內得分加倍。
6.  **遊戲結束**：計時結束遊戲結束，分數會提交至全球排行榜。

## 特色功能

*   **掉落物品遊戲**：經典街機風格玩法。
*   **測驗系統**：通過一系列多選題測試對媛的了解。
*   **多語言支援**：遊戲介面及問題提供繁體中文 (`zh-TW`)及英語 (`en`)版本。
*   **Firebase整合**：
    *   **全球排行榜**：所有玩家分數匯總，共同達成全球里程碑。
    *   **個人成就**：玩家能累積分數以解鎖個人獎勵。
    *   **進度持久化**：玩家進度使用瀏覽器中的唯一匿名用戶ID保存在本地存儲。
*   **特別生日活動**：若在媛的生日（1月5日）玩遊戲，將顯示特別訊息及主題。
*   **動畫角色**：玩家角色具備待機和反應動畫。

## 技術架構

*   **前端**：HTML5、CSS3、JavaScript (ES6)
*   **後端/資料庫**：Firebase (Firestore) 用於分數及玩家資料存儲。
*   **資產**：包含角色與物品圖片，以及遊戲事件音效。

## 本地執行方式

1.  複製或下載此專案。
2.  在現代瀏覽器（如Chrome、Firefox或Edge）中打開`index.html`檔。
3.  遊戲將載入並可開始遊玩。Firebase功能（排行榜與里程碑）需要網路連線。
